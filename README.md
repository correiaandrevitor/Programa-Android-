# Aplicativo de Orçamentos

O Nosso Protejo é um aplicativo mobile desenvolvido com React Native que tem como objetivo simplificar o processo de cálculo e geração de orçamentos para etiquetas comercializadas por milheiro. A aplicação foi projetada para atender empresas gráficas e fabricantes de etiquetas que necessitam realizar cotações rápidas, precisas e padronizadas diretamente pelo smartphone ou tablet.

Em muitos cenários do mercado gráfico e de impressão, o cálculo de preços para etiquetas envolve diversas variáveis, como dimensões, quantidade solicitada, tipo de material, acabamento, custos de produção, margem de lucro e outros fatores que impactam diretamente no valor final do orçamento. Realizar esses cálculos manualmente pode gerar erros, inconsistências e perda de produtividade.

Pensando nisso, este aplicativo foi desenvolvido para oferecer uma solução moderna, intuitiva e eficiente, permitindo que vendedores e operadores realizem orçamentos em poucos segundos, reduzindo falhas humanas e aumentando a agilidade no atendimento ao cliente.
